Detailed Description

template<typename T>
class qtpyt::QPySharedArray< T >

forward declaration for external usage in other headers

A shared data container that can be treated as a vector of T on the C++ side and as a memoryview the Python side.

Template parameter T is the element type.

Key features:

  • Supports wrapping external buffers without copying.
  • Optionally takes ownership of external buffers or keeps an external owner alive via a std::shared_ptr<detail::OwnerState>. Preserves buffer lifetime across C++/Python boundary.
  • Provides detach() semantics: modifying operations create a private copy.
