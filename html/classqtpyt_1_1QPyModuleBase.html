Detailed Description

Core functionality for building a Python module and invoking functions.

Instances own references to Python objects (pybind11::object) representing a module and optionally a selected callable within that module. The concrete module creation is controlled by QPySourceType.

Copy/move: Copy and move operations are disabled due to Python object semantics and internal state that should not be duplicated.
