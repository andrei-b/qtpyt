Detailed Description

Public C++ interface for invoking Python module functions via qtpyt.

QPyModule represents a Python "module" (from source text or file) that can be executed and interacted with from C++ using Qt types. It supports synchronous module setup (done in the base) and asynchronous function invocation returning QPyFuture.

Threading model:
- Instances track the creating thread id and expect interactions to respect the underlying Qt/Python thread constraints.
- Asynchronous calls are scheduled via qtpyt thread facilities (see qpythreadpool.h).

Note
PYBIND11_NO_KEYWORDS is defined to avoid keyword argument collisions in pybind11.
