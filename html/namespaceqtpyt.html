◆ QPyRegisteredType

using qtpyt::QPyRegisteredType = typedef std::variant<QMetaType::Type, QString, QMetaType>

A "registered type" used to guide conversion between Python and Qt. Any (almost) Qt metatype can be specified using this variant.

The variant supports specifying a type as: - QMetaType::Type (built-in Qt metatype id) - QString (type name) - QMetaType (full metatype object)
